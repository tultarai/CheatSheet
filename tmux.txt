■ターミナル
tmux list-session  セッション一覧を表示 (tmux ls)
tmux attach -t     セッションへアタッチする (-tは省略可能)
tmux rename -t     セッション名を変更する
tmux kill-session  セッションを削除する

tmux new-session   新規セッションを開始する

tmux list-client   クライアント一覧を表示

■でタッチ
C-j d       デタッチする

■ウィンドウ操作
C-j c       新規ウィンドウを作成
C-j n       次のウィンドウへ移動
C-j p       前のウィンドウへ移動
C-j [数字]  数字で指定したウィンドウへ移動
C-j w       現在のウィンドウ一覧を表示

■ペイン操作
C-j q       ペイン一インジケータ表示 (表示中に数字を押すと該当ペインへ移動)
C-j v       垂直に分割
C-j s       水平に分割

C-j o       次のペインへ移動
C-j h       左のペインへ移動
C-j j       下のペインへ移動
C-j k       上のペインへ移動
C-j l       右のペインへ移動

C-j <       ペインのサイズ変更
C-j >       ペインのサイズ変更
C-j -       ペインのサイズ変更
C-j +       ペインのサイズ変更

C-j [space] ペインの配置を変更 (お勧めできない)

■コピーモード
C-j [       コピーモードに入る
Space       コピーの始点を選ぶ
Return      コピーの終点を選ぶ
C-j ]       貼り付ける

■etc
C-j h       Help
C-j :       Command Prompt
